# Random Quote Generator

A simple JavaScript app that generates quotes using both local data and an external API.

## Features
- Random quote generation using Math.random()
- API-based quote fetching using Fetch API
- Error handling for API failures
- Responsive and clean UI

## Tech Stack
- HTML
- CSS
- JavaScript
- REST API (Fetch)
